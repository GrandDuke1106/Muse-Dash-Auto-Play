# Muse Dash Auto Play（梦游少年计划）
傻瓜式操作，只要你第一个键打准了就能全连大部分歌，AP部分歌。如果某首歌一直搞不定自己去改代码。

因为我只是为了Stream成就，所以我没仔细研究铺面文件结构，有些铺子没法正常使用是正常的。
